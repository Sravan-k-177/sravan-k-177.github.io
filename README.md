# Portfolio Website

A John Wick themed portfolio website built with HTML, CSS, and JavaScript.

# Features

- **Continental theme** — colour aesthetic inspired by the John Wick universe
- **Animated loader** — Continental seal with a progress bar on entry
- **Interactive cursor** — custom crosshair cursor with a trailing ring
- **Click effects** — bullet holes, blood splatter, and gold coins on every click
- **Typewriter effect** - different roles are shuffled
- **Skill bars** — progress bar with animation that triggers on scroll
- **Responsive design** — works on mobile and desktop
- **Contact form** — powered by [EmailJS](https://www.emailjs.com/), messages are delivered directly to my inbox with success/failure feedback — without a backend.

## Tech Stack

- HTML5, CSS3, JavaScript
- EmailJS (contact form)
- Google Fonts — Cinzel, Rajdhani, Share Tech Mono (Matches the John Wick universe)
